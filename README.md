# TelecomCustomerChurn
The churn dataset contains data on a variety of telecom customers and the modeling challenge is to predict which customers will cancel their service (or churn).

A case study from the "Machine Learning Toolbox" exercise on Datacamp.

Points to Note:
**Goals of the Caret package**
1. Simplify the predictive modeling process.
2. Make it easy to try many models and techniques.
3. Provide common interface to many useful packages.
